# Instrucciones para Configurar el Proyecto

## 1. Instalar Dependencias
Ejecuta el siguiente comando para instalar todas las dependencias necesarias:
```bash
npm install
```

## 2. Instalar Dependencia de Facturapi (opcional)
En caso de requerir la dependencia `facturapi`, instálala con el siguiente comando:
```bash
npm install facturapi
```

## 3. Configurar Llaves de Amazon
Asegúrate de implementar las llaves proveídas por Amazon. Consulta el Manual de integración de APIs externas para evitar problemas relacionados con el uso público de las llaves.

## 4. Inicializar el Proyecto
Para inicializar el proyecto, ejecuta:
```bash
node index.js
```
